# InstaScripts #
###### A Collection of Instagram Facilitator Scripts ######

### [login.js](login.js) ###
```shell
node login
```

### [follow.js](follow.js) ###
```shell
node follow [user]...
```

### [unfollow.js](unfollow.js) ###
```shell
node unfollow [user]...
```

### [requests.js](requests.js) ###
```shell
node requests [--unfollow]
```

### [user.js](user.js) ###
```shell
node user [--login] [user]
```
